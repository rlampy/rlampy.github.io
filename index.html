The Class

This course covers basic web development fundamentals and e-commerce trends. Specific topic coverage includes: electronic commerce, technology infrastructure, & web site design and creation.

The project

Students will design, develop, and publish a web site using recommended design practices. Completed sites will be presented to the class; this project (with all written submissions) constitutes the final exam.

Grading

This project will be graded through a series of milestones designed to make this a semester-long project. This process will include planning and design, as well as a written report on the completed project.

Planning and Design

Students will create a website of their own design, reflecting their own interests. The final website will be required to have a home page with four or five subpages.

Other Requirements

Students are required to use an external style sheet to configure most text, color, and page layout. They must publish the project via www.x10host.com. Exact requirements are given in the milestone assignments.
